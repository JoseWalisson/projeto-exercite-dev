Exercício: Organize os textos colocando as tags corretas

Instruções:

  1. Abra o código HTML fornecido no editor de texto de sua preferência.
  2. Neste exercício, o objetivo é organizar os textos de forma hierárquica, utilizando as tags HTML apropriadas para títulos e parágrafos.
  3. Identifique os títulos e subtítulos no texto fornecido. Eles são as seções que devem ser destacadas de forma hierárquica.
  4. Utilize as tags HTML <h1>, <h2>, <h3>, etc., para representar os títulos em ordem hierárquica. A tag <h1> representa o título principal, e as demais tags representam os subtítulos em ordem de importância.
  5. Adicione as tags HTML corretas para os títulos e parágrafos de cada seção. Por exemplo, utilize a tag <h1> para o título "Encontre seu carro dos sonhos!" e a tag <p> para os parágrafos relacionados a esse título.
  6. Organize todos os parágrafos em suas respectivas seções, de acordo com os títulos e subtítulos identificados.
  7. Verifique se todas as tags de abertura têm suas respectivas tags de fechamento para evitar erros no código.
  8. Após concluir a organização hierárquica dos textos, salve o arquivo HTML
  9. Abra o arquivo HTML em um navegador para verificar se os títulos e parágrafos estão estruturados corretamente e se a hierarquia está claramente visível na página.
  10. Pratique quantas vezes forem necessárias para se familiarizar com a organização hierárquica dos textos em HTML.
Baixar exercício!

Atenção: Lembre de abrir esse arquivo em seu editor de código preferido!
